Usage:

    $ rester <filename>

Arguments:

    filename - A Restfile

Options:
    --insecure [default: false] - do not validate SSL certificate (macOS only)
    --count - number of iterations to loop for (implies `--loop 0`)
    --duration - duration <seconds> to loop for (implies `--loop 0`
    --loop - keep executing file every <loop> seconds
    --stats [default: false] - Show stats
    --timeout - Request timeout
    --verbose [default: false] - Verbose output
    --workdir [default: ] - Working directory (for the purpose of resolving relative paths in Restfiles)
